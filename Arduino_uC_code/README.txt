download arduino IDE from here
http://arduino.cc/en/Main/Software
use arduino-1.0.5 or arduino-1.0.6

copy XBee,DHT, Debug,PinChangeInt,eHealth folders to arduino-1.0.x/libraries

For the Arduino LilyPad USB + accelerometer+xbee use folder lilypad_acc_code
For the Arduino Uno + environmental sensors + xbee use folder uno_environmental_uC_code
If using only PIR, Humidity, Temperature sensors upload uno_environmental_PIR_HUM_TEMP_MQ2_Door to the arduino
If using only PIR, Humidity, Temperature, Gas Mq2, Door sensors upload uno_environmental_PIR_HUM_TEMP_MQ2_Door to the arduino
